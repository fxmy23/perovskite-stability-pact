## R1: 高 IF 论文如何处理算法? PCRL 能否精进?

### 1.1 高 IF 论文的算法创新模式 (2024-2025)

调研发现 5 大创新方向:

| 方向 | 代表工作 | 创新本质 | 我们能否做 |
|------|----------|----------|-----------|
| **等变神经网络** | PAGNN (Adv Opt Mater 2025) | E(3)对称性嵌入 | ❌ 需GPU+结构数据 |
| **几何感知协同注意力** | Solar-GECO (NeurIPS 2025) | 组成+几何联合推理 | ❌ 需结构数据 |
| **多保真度学习** | Mannodi group | 融合廉价+昂贵DFT | ⚠️ 需多源数据 |
| **物理约束残差** | PINN (Nat Sci Rep 2025) | 残差受PDE约束 | ✅ **我们正在做** |
| **基础模型/通用势** | M3GNet lineage | 迁移表示 | ❌ 需大规模预训练 |

**关键洞察**: 我们的 PCRL 属于"物理约束残差"方向——这是**唯一不依赖GPU/结构数据的创新路径**,非常适合我们的约束(笔记本+组成特征)。

### 1.2 PCRL 当前的不足 (诚实)

当前 PCRL (penalty=0.1):
- 形成能 R²=0.9062 vs 纯ML 0.9097 → **略低于纯ML**
- 物理约束的"精度代价"虽小但存在

**问题本质**: 我们的物理约束是"软约束"(feature_penalty 降低增益),
不如 PINN 的"硬约束"(损失函数显式项)。审稿人可能质疑"约束太弱,几乎是标准PGML"。

### 1.3 ★ PCRL 精进方案 (三个层次, 逐级提升)

#### 层次1: 强化物理约束的数学严谨性 (必做)
当前 feature_penalty 是启发式。应改为**显式损失项**:
$$\mathcal{L} = \|y - f - g\|^2 + \lambda \cdot \text{SHAP}_{phys}(g)^2$$
用 SHAP 值量化残差模型对物理特征的依赖, 作为正则项。
**这把"软约束"升级为"有理论基础的约束", 论文方法章节更有说服力。**

#### 层次2: 多保真度物理基线 (推荐)
当前物理基线只用 Goldschmidt 判据。可以**叠加多个物理模型**:
- 容忍因子 t (Goldschmidt 1926)
- 八面体因子 μ
- **新息参量** τ = (r_A/r_B)·(χ_A/χ_B) (我们自己定义的复合判据)

多物理基线投票/加权, 提升物理基线 R² (当前 0.71→目标 0.75+)。

#### 层次3: 不确定性感知的物理-ML 切换 (冲二区)
在物理可信区(t∈[0.8,1.0])用物理, 在物理失效区用ML:
$$\hat{y} = w(\mathbf{x}) \cdot f_{phys} + (1-w(\mathbf{x})) \cdot g_{ML}$$
其中 $w(\mathbf{x})$ 由物理基线的不确定性决定。
**这是真正的"物理引导", 而非简单残差。**

## R2: 评判指标体系 — 我们缺什么?

### 2.1 Matbench Discovery 的完整指标体系 (官方)

调研 [Matbench Discovery](https://matbench-discovery.materialsproject.org/) 发现,
稳定性预测的标准指标远不止 RMSE/MAE/R²:

| 指标 | 定义 | 我们有吗 | 重要性 |
|------|------|----------|--------|
| RMSE/MAE/R² | 回归精度 | ✅ 有 | 基础 |
| **F1 Score** | 稳定/不稳定分类的F1 | ❌ **缺** | ⭐⭐⭐ |
| **Precision** | 预测稳定中真稳定比例 | ❌ **缺** | ⭐⭐⭐ |
| **Recall (TPR)** | 真稳定中被找出的比例 | ❌ **缺** | ⭐⭐⭐ |
| **FPR** | 假阳性率 | ❌ **缺** | ⭐⭐ |
| **DAF** (Discovery Acceleration Factor) | 发现加速因子 | ❌ **缺** | ⭐⭐⭐ |
| **AUC-ROC** | 分类判别能力 | ❌ **缺** | ⭐⭐ |
| **PICP/MPIW** | 不确定性校准 | ✅ 有 | ⭐⭐ |

### 2.2 ★ 为什么这些指标关键 (审稿人视角)

**问题**: 我们当前只报 R²=0.80 (凸包能)。审稿人会问:
> "R²=0.80 看起来还行, 但你的模型**判断一个材料是否稳定**的准确率是多少?
> 在 100 个预测稳定的材料里, 有多少真的稳定? (Precision)
> 在 100 个真正稳定的材料里, 你找出了多少? (Recall)"

**这正是 Matbench Discovery 强调 F1/Precision/Recall 的原因**——
材料发现是**分类问题**(稳定/不稳定), 不只是回归问题。

**DAF (Discovery Acceleration Factor)** 更关键:
> "用你的模型筛选, 比随机猜测能快多少倍发现稳定材料?"
DAF = (模型找出的稳定材料密度) / (全空间稳定材料密度)。
M3GNet 的 DAF=2.70 是业界标杆。

### 2.3 ★ 我们的指标补充方案 (必做)

把稳定性预测重构为**分类任务** (E_hull < 0.05 = 稳定):
1. 用模型预测 E_hull, 阈值分类
2. 计算 Precision/Recall/F1/AUC-ROC
3. 计算 DAF (在 top-N 候选中的稳定富集度)
4. 画 PR 曲线 + ROC 曲线

**这些指标会让论文的"材料发现"部分从定性变为定量**, 大幅提升说服力。

## R3: 实用性 — 审稿人说的"重要参考价值"

### 3.1 npj 审稿标准第3条: "important to the specific field"

调研发现, 高实用性论文的共同特征:

| 实用性要素 | 说明 | 我们有吗 |
|-----------|------|----------|
| **代码开源** | GitHub + 可复现 | ⚠️ 有代码但未整理 |
| **数据可获取** | 公开数据集+处理脚本 | ✅ wolverton公开 |
| **用户工具** | Web app / GUI / API | ❌ **缺** |
| **实验验证** | 合成+表征预测的候选 | ❌ 无法做(纯计算) |
| **适用域明确** | 告诉用户何时可信 | ✅ LOEO做了 |
| **新材料发现** | 真实候选清单 | ✅ 9个新候选 |

### 3.2 ★ 实用性提升方案 (三个杠杆)

#### 杠杆1: 候选材料数据库 (强烈推荐, 低成本高收益)
把 9 个新候选 + 147 个反常材料整理成**结构化数据库**,
附预测值/不确定性/物理特征, 作为论文的**补充材料**。
审稿人和读者可以直接拿去用。这是"重要参考价值"的直接体现。

#### 杠杆2: 简单的预测工具 (推荐)
写一个 `src/predict.py`: 输入化学式(如 "LaTiO3"), 输出预测的 E_f/E_hull/稳定性/不确定性。
几十行代码, 但让论文从"学术研究"变成"可用工具"。

#### 杠杆3: 适用域指南 (推荐)
基于 LOEO 结果, 明确写出:
> "本模型适用于 A 位为稀土/碱土、B 位为过渡金属的钙钛矿;
> 对含 Ac/Pb/Al 的体系预测可靠性低, 建议额外验证。"
这是负责任的工程实践, 审稿人尊重这种诚实。

## 审慎判断: 整体评估与优先级

### 答你的核心问题

**Q: PCRL 能否继续精进做到出彩?**
→ 能。层次1(显式SHAP约束)+层次2(多物理基线)可以把PCRL从"标准PGML变体"提升为"有理论创新的方法"。但层次3(不确定性切换)是冲二区的, 工作量大。

**Q: 评判指标我们缺什么?**
→ **缺关键的 F1/Precision/Recall/DAF**。这是 Matbench Discovery 的标准, 审稿人必看。补充这些是**性价比最高的改进**。

**Q: 实用性够不够"重要参考价值"?**
→ 当前中等。加候选数据库+预测工具+适用域指南后, 实用性显著提升。

### ★ 改进优先级排序 (投入产出比)

| 优先级 | 改进 | 工作量 | 收益 | 推荐度 |
|--------|------|--------|------|--------|
| **P0** | 补充F1/Precision/Recall/DAF指标 | 0.5天 | ⭐⭐⭐ | **必做** |
| **P0** | 候选材料数据库(补充材料) | 0.5天 | ⭐⭐⭐ | **必做** |
| P1 | PCRL层次1: 显式SHAP约束 | 1天 | ⭐⭐⭐ | 强烈推荐 |
| P1 | 预测工具 predict.py | 0.5天 | ⭐⭐ | 推荐 |
| P2 | PCRL层次2: 多物理基线 | 1天 | ⭐⭐ | 可选 |
| P2 | 适用域指南文档 | 0.5天 | ⭐⭐ | 推荐 |
| P3 | PCRL层次3: 不确定性切换 | 2天 | ⭐⭐⭐ | 冲二区再做 |

### 一句话结论

> **补充 F1/DAF 等分类指标 + 候选数据库 = 最高性价比的"出彩"路径。**
> 这两步让论文从"回归预测"升级为"发现工具", 实用性和创新性同时提升。
> PCRL 的 SHAP 约束强化是方法层面的进一步精进。
