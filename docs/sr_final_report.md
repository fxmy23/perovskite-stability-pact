## 二、SR 创新的真实价值 (诚实, 非夸大)

**SR 物理层做到了什么 (真)**:
1. ✅ 把物理贡献从黑盒 R²=0.789 翻译成**显式方程** (涉及 a_site_en 电负性、radius_ratio、b_site_group)
2. ✅ 总精度无损 (0.9103 ≈ KRR 0.911)
3. ✅ conformal/AD 在 SR 物理层下全部成立
4. ✅ 发现"SR 适用边界"判据 (SNR/线性度) — 独立科学贡献

**SR 物理层没做到什么 (诚实承认)**:
1. ⚠️ **没有提升精度** — 纯 ML (0.914) 反而略高于 SR+ML (0.910)
2. ⚠️ **物理层独立 R² (0.41) 远低于 KRR (0.79)** — 可解释性有代价
3. ⚠️ **5 折公式结构不一致** — SR 解非唯一 (已知特性)
4. ⚠️ **凸包能上完全失败** — 只能用于形成能

## 四、SR 公式的物理可读性 (debug D3 输出)

替换 X→物理量后, fold1 形成能公式涉及:
- **a_site_en** (A 位电负性) — 反复出现, 嵌套 sqrt, 说明 A-B 电负性差主导形成能
- **radius_ratio_AB** (半径比) — 几何匹配贡献
- **b_site_group** (B 族数) — 电子结构贡献

这符合化学直觉: **形成能 ≈ 化学键强度, 由电负性差 + 几何匹配 + 电子结构决定**。
虽然公式嵌套较深 (sqrt 套 sqrt), 但核心物理量清晰。

## 六、剩余诚实局限 (论文必须讨论)

1. SR 公式非唯一 (5 折不同结构) — 建议报"公式族"而非单一公式
2. SR 物理层 R²=0.41 < KRR 0.79 — 可解释性代价, 残差补足
