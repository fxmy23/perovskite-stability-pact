### Q2: 结果是否 100% 可信度?

**没有任何 ML 科研结果是 100% 可信的——这是科学本质, 不是项目缺陷。** 诚实的可信度分层:

| 结论 | 可信度 | 依据 |
|---|---|---|
| R²=0.910 (formation) | **高** | 5-seed mean±std 一致, bootstrap CI [0.903,0.917], 无泄露 |
| PICP=0.816±0.005 (conformal) | **高** | 5-seed 全达标, 200 次合成试验均值 0.805, 理论保证 |
| σ-误差相关 r=0.389, p=1e-177 | **高** | 极显著, 不确定性真实有意义 |
| DAF=6.11, AUC=0.926 | **高** | 独立验证公式正确, base rate 9.4% 合理 |
| 物理贡献 78.9% + ML增量 12.1% | **中高** | 定义明确, 但"贡献"本质是 R² 分解 (有解释空间) |
| 候选 LaAlO3 稳定 | **中** | matbench 独立 DFT e_form=-0.10 佐证, 但 LOEO 严格测试 0 通过 |
| 跨数据集泛化 | **不可证** | ★重要发现: matbench与wolverton形成能不同量纲(Pearson 0.358,偏差2.5eV),数值不可直接互证 |
| 模型间差异显著 | **低** | Wilcoxon 5-seed 最小 p=0.0625, 无法区分 LightGBM/XGBoost/RF |
| LOEO 外推结论 | **中** | 单 CV 切分, 元素频率不均 (Ho 136 样本 vs Ac 137) |

**最大不确定性来源**: (1) Wilcoxon 在 5 seed 下无分辨率 (这是统计极限, 非项目可修);
(2) 候选零 DFT 验证 (用 matbench 收录核对弥补, 但两库量纲不同, 仅证存在性非数值一致);
(3) 数据源单一 (仅 wolverton_oxides 一个组的高通量)。

**★ 审计中发现并修正的重大问题 (诚实记录)**:
原 V2 声称"LaAlO3 matbench e_form=-0.10 独立佐证稳定"。审计跨库量纲诊断发现
matbench 与 wolverton 形成能**不同量纲** (Pearson r=0.358, 系统偏差 +2.5 eV),
因此该数值佐证**无效**。已修正 candidate_validation: matbench 仅证"化合物被独立
DFT 收录", 不证数值一致。候选验证等级因此下调 (0 strong, 2 moderate)。
这是审查的真正价值——发现并修正了自证式过度声明。

## 二、当前真实水平定位 (对标 2024-2025 模范论文)

### 2.1 强项 (真正超过多数三区论文的)

1. **统计严谨性**: 5-seed + bootstrap CI + Wilcoxon + σ-error p 值。
   多数三区 ML 论文只报单次 R², 我们报 mean±std + CI + 显著性。
2. **不确定性有理论保证**: conformal PICP 边际保证成立 (vs 多数论文的 ensemble σ 启发式)。
3. **LOEO 全 73 元素外推**: 三区钙钛矿 ML 论文里罕见, 真正差异化。
4. **应用域多方法对照**: σ/kNN/leverage 三方法, 非单启发式。
5. **候选独立验证**: LaAlO3 在 matbench 独立 DFT 中 e_form=-0.10。
6. **诚实**: PCRL 负面结果 + phys_share 重表 + LOEO 局限标注。

### 2.2 弱项 (低于或匹配三区模范的)

1. **R²=0.910 低于 2025 SOTA**: MDPI Materials 2025 = 0.928, AI Chemistry 2025 >0.965。
   我们靠差异化 (conformal+LOEO+AD) 而非 R² 竞争。
2. **数据源单一**: 仅 wolverton_oxides (4914)。模范论文多用 MP/OQMD/AFLOW。
3. **无 DFT 验证**: 用 matbench 交叉核对 (LaAlO3 e_form<0) 是强但非完全等价。
4. **Wilcoxon 无分辨率**: 5-seed 下 p≥0.0625, 无法声明"LightGBM 显著优于 RF"。
5. **PCRL 失败**: 虽诚实报告, 但"方法创新"因此只剩 PACT (单点而非多点)。

### 2.3 与三区录用线的距离

**Computational Materials Science (目标期刊, IF≈3.3, 三区)** 典型录用论文要求:
- ✅ 完整数据 + 可复现代码 — 我们有
- ✅ 多模型对比 + mean±std — 我们有
- ✅ 不确定性/可解释性 — 我们有 (conformal + SHAP)
- ⚠️ 候选材料 + 验证 — 我们有交叉核对, 缺 DFT
- ⚠️ 与公开数据库对比 — 缺 (MP API 不可达)
- ✅ 诚实 + 严谨 — 我们有

**判断**: 在"不确定性量化 + 应用域 + 外推评估"这三个维度, 我们**超过**多数三区论文;
在"R² 竞争力 + 数据多样性 + DFT 验证"上**低于**模范。综合**处于三区录用线附近**。
