# 符号回归适用边界分析 (SR Applicability Boundary)

**这是本项目独有的科学贡献**——定量解释为什么符号回归在某些材料性质上成功、某些失败。

## 一、问题与方法

**问题**: 我们用 gplearn 符号回归(SR)在 14 个物理特征上分别预测形成能与凸包能(E_hull),
发现 SR 在形成能上 R²=0.47 (CV), 但在凸包能上仅 R²=0.19 且不稳定。为何?

**方法**: 6 个角度交叉验证——信噪比、线性度、分布形态、单特征解释力、SR收敛诊断、
ML特征重要性。

## 二、核心结论

形成能与凸包能的根本差异**不是"难度大小",而是"物理本质不同"**:

| 性质 | 形成能 E_f | 凸包能 E_hull |
|---|---|---|
| 物理含义 | 化合物 vs 单质的能量差 | 化合物 vs 所有竞争相的能量差 |
| 主导因素 | **化学键强度** (电负性、电子结构) | **几何匹配** (容差因子、半径比) + 相竞争 |
| 与特征关系 | 近**线性单调** | **非单调钟形** (t≈0.9 最优, 过大过小都不稳定) |
| SR 可表达性 | 加减乘除可表达 | 钟形关系需 sin/exp, 超出 basic 运算符 |

## 三、6 重证据

### 证据1: 信噪比 (SNR)
- 形成能 SNR = **2.61** (物理规律强, 噪声占比低)
- 凸包能 SNR = **0.94** (噪声占主导, 物理规律被淹没)
- KRR 物理层可解释方差: 形成能 72.3% vs 凸包能 48.6%

### 证据2: 线性度
- 形成能线性模型 R² = **0.567** (近线性, SR 简单运算符可拟合)
- 凸包能线性模型 R² = **0.290** (强非线性, SR basic 运算符力不从心)

### 证据3: 分布形态
- 形成能: 偏度 0.315 (近对称), 峰度 -0.314 (平顶) → 规整连续分布
- 凸包能: 偏度 **1.005** (强右偏), 峰度 **1.142** (尖峰) → 两极分化
  (9.4% 样本 <0.05 近稳定, 60.4% >0.5 明显不稳定)

### 证据4: 单特征解释力差异
- 形成能主导特征: b_site_en (|r|=0.29), b_site_group (0.30), a_site_en (0.54)
  → **电负性与电子结构**, 这类关系 SR 乘除可表达
- 凸包能主导特征: a_site_radius (|r|=0.51), tolerance_factor (0.46)
  → **几何因子**, 容差因子本身是比值 t=(r_A+r_O)/(√2(r_B+r_O)),
  且 E_hull 与 t 是**钟形**关系, SR basic 运算符难表达

### 证据5: SR 收敛诊断
- 凸包能预测常数(中位数0.665)的 R²=-0.017, **反超某些 SR 配置**
  → gplearn 遗传算法"发现"返回常数比拟合更省事, 塌缩为常数 (fold5 的 0.655)
- 形成能预测常数 R²=-0.005, SR 配置均显著优于常数, 无塌缩

### 证据6: ML 特征重要性
- 形成能 LightGBM: 物理特征占重要性 28.9% (15个特征)
- 凸包能 LightGBM: 物理特征占 30.0%
- → ML 已充分吸收物理信息, 独立物理层(含SR)的价值是"可解释性"非"提精度"

## 四、对符号回归方法学的贡献

本分析首次(据我们所知)定量给出 SR 在材料性质预测中的**适用判据**:

> **SR 适用的必要条件**: (1) SNR ≥ ~2; (2) 线性模型 R² ≥ ~0.5;
> (3) 目标-特征关系单调或简单非线性; (4) 分布无强尖峰/两极分化。
> 形成能满足全部 → SR 成功; 凸包能违反 (1)(2)(4) → SR 失败。

这个判据可指导未来研究**预判 SR 是否适用于某材料性质**, 避免盲目尝试。

## 五、诚实局限

1. SR 公式 5 折结构不一致 (非唯一解) — 符号回归的已知特性, 论文应讨论。
2. 根因分析基于相关性 (Pearson |r|), 非因果; 但 SNR/线性度的解释有理论支撑。
3. "SNR≥2" 阈值是经验观察 (基于 2 个目标), 严格阈值需更多性质验证。
