# 模型适用域指南 (Applicability Domain)

> 本文档基于 LOEO 外推测试结果, 明确 PCRL 模型的可信使用范围。
> 这是负责任的工程实践, 也是 npj Computational Materials 审稿标准
> 第3条("important to field")的体现。

---

## 一、模型适用条件

### ✅ 高可信区域 (LOEO R² > 0.85)

以下元素的钙钛矿, 模型预测高度可信:

| 元素类型 | 代表元素 | LOEO R² (形成能) | 说明 |
|----------|----------|-----------------|------|
| 稀土(轻) | La, Pr, Nd | 0.93-0.96 | 训练集覆盖充分 |
| 稀土(重) | Dy, Ho, Tm, Yb | 0.89-0.96 | 化学性质典型 |
| 过渡金属(第一行) | Ti, V, Cr, Mn, Fe, Co, Ni | 0.85-0.94 | 钙钛矿常见B位 |
| 碱土 | Ca, Sr, Ba | 高 | 经典A位 |

**建议**: 含上述元素的钙钛矿, 预测可直接用于筛选。

### ⚠️ 中等可信区域 (LOEO R² 0.40-0.85)

| 元素 | LOEO R² | 建议 |
|------|---------|------|
| Cu, Zn, Ga, Ge | 0.50-0.80 | 预测仅供参考, 建议补充验证 |
| 碱金属(K, Rb, Cs) | 0.40-0.70 | A位碱金属钙钛矿少, 谨慎使用 |

### ❌ 低可信区域 (LOEO R² < 0.40, 不可外推)

以下元素的钙钛矿, **模型预测不可靠**, 必须额外验证:

| 元素 | LOEO R² (形成能) | 原因 |
|------|-----------------|------|
| **Ac (锕)** | 0.33 | 锕系, 训练样本化学空间稀疏 |
| **Pb (铅)** | 0.22 | 重主族元素, 电子结构特殊 |
| **Al (铝)** | 0.25 | 轻元素, 钙钛矿中非典型B位 |
| **Os (锇)** | 0.27 | 5d过渡金属, 训练不足 |
| **Hg (汞)** | 0.36 | 液态金属, 化学异常 |

**建议**: 含上述元素的体系, 模型预测仅供探索性参考, **不可用于决策**,
必须配合 DFT 验证或实验确认。

## 二、可信度量化指标

除了元素类型, 用户可通过以下指标判断单次预测的可信度:

1. **不确定性 (集成方差 std)**:
   - std < 0.005: 高可信 (对应候选库 Confidence=High)
   - std 0.005-0.01: 中可信
   - std > 0.01: 低可信, 谨慎

2. **容忍因子 t 落入经典区**:
   - t ∈ [0.8, 1.0] 且 μ ∈ [0.414, 0.732]: 物理支持, 可信度加成
   - 不在经典区: 物理判据不支持, 需更谨慎

3. **DAF 富集度** (批量筛选时):
   - 在 top 10% 候选中, 稳定材料富集 6.4 倍
   - 越靠前的候选越可能稳定

## 三、已知局限

1. **仅适用 ABO₃ 单钙钛矿氧化物**: 双钙钛矿(A₂BB'O₆)未充分验证
2. **组成特征 only**: 不使用晶体结构 (排除 DFT 泄露), 精度上限受限
3. **wolverton 数据源**: 跨数据集(如 matbench)泛化性有限 (Spearman=0.41)
4. **DFT 设置**: 预测基于 PBE 泛函, 与实验值有系统偏差

## 四、推荐使用流程

```
输入化学式 (如 LaTiO3)
    ↓
predict.py 输出 Ef / Ehull / 不确定性 / 物理特征
    ↓
检查适用域:
  - 元素是否在高可信区? (查上表)
  - 不确定性 std < 0.01?
  - t 是否在经典区?
    ↓
全部 yes → 高可信, 可用于筛选
部分 no  → 中可信, 建议补充 DFT 验证
全部 no  → 不可信, 必须独立验证
```
