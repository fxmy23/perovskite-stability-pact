## 〇、先纠正一个重要结论 (LOEO 实验改变评判)

**作者之前说**: "纯 ML (0.914) 反超物理层, 物理层没提精度。"
**审稿人质疑**: 这是在 CV 插值场景评估的——物理层的真正价值是外推, 你在错误场景评判。

**判决实验结果** (留 5 个 A 位元素, _loeo_judgement.py):

| 方法 | CV 插值 R² | LOEO 外推 R² (5元素平均) | 外推跌幅 |
|---|---|---|---|
| 纯 ML | 0.914 | **0.373** | -0.541 |
| KRR+ML | 0.910 | 0.381 | -0.529 |
| **SR+ML** | 0.910 | **0.422** | -0.488 |

**判决**: 在 LOEO 外推下, **SR+ML (0.422) > 纯 ML (0.373), 领先 0.049**。
物理层的价值确实在外推——CV 下纯 ML 反超, 但外推下 SR+ML 跌得更少。
**"物理层没提精度"是 CV 场景的伪结论, 必须修正。**

(诚实: KRR+ML 只领先纯 ML 0.008, 几乎可忽略; SR+ML 领先 0.049 更显著,
但 B 元素上所有模型崩溃 R² 全负, 绝对值仍偏低。)

## 一、问题 ①-④ 的逐个严苛审视 + 解决方案

### 问题①: 物理层没提精度 (纯ML反超)
**根因**: CV 插值下 ML 已吸收物理信息。
**新证据**: LOEO 外推下 SR+ML 反超纯 ML (上表)。
**解决**: ✅ **已在本次审稿中用 LOEO 实验解决**——论文应同时报 CV 和 LOEO,
强调"物理层提升外推稳健性而非插值精度"。

### 问题②: SR 物理层独立 R²=0.41 远低于 KRR 0.79
**根因**: SR 受限于简单运算符 (加减乘除根号), 表达力 < RBF 核。
**调研发现的解决方案**:
- **(A) 丰富 function_set**: 加入 sin/exp/tanh 可表达钟形关系 (容差因子最优值)。
  风险: 公式可解释性下降。
- **(B) 蒸馏**: 先训 KernelRidge, 用其预测作 SR 的软目标, 引导 SR 逼近核函数。
  中等难度。
- **(C) 接受**: SR 物理层 R²=0.41 是可解释性的合理代价, ML 残差补足。
**解决**: 推荐先试 (A) 丰富算子 + 帕累托选择, 预期 R² 0.41→0.55-0.65。
若仍不够, 用 (C) 接受 + 诚实报告。

### 问题③: 5 折公式不一致 (非唯一解)
**根因**: 符号回归的已知特性 (ACM Computing Surveys 2024 明确讨论)。
**调研发现的解决方案**:
- **符号集成 (Symbolic Ensemble)**: 跑 N 次 SR, 对公式"投票"——统计每个物理量
  在 N 个公式中出现的频率, 报"特征共识度"而非单一公式。
  (PySR 社区讨论 #273 有先例)
- **帕累托前沿模型选择** (Bayesian SR, Roy. Soc. 2025): 在复杂度-精度帕累托前沿上
  用稳定性准则选公式。
**解决**: 推荐**符号集成**——跑 10 次 SR, 报"X7(a_site_en) 在 10/10 公式中出现,
X10(b_site_group) 在 8/10 出现"作为共识, 而非纠结单一公式。直接解决非唯一性。

### 问题④: 凸包能 SR 失败
**根因**: 凸包能 SNR=0.94 + 强非线性 + 钟形关系, SR basic 算子无法表达。
**调研发现的解决方案**:
- **(A) 加入 sin/square 算子**: 钟形关系可用 Gaussian/sin 近似。
  风险: 凸包能本身噪声大, 即使丰富算子 R² 可能仍低。
- **(B) 转分类任务**: 凸包能的核心应用是"稳定/不稳定"二分类。
  SR 对分类 (阈值化) 可能比回归更鲁棒。
- **(C) 接受 + 适用边界分析**: 把"凸包能 SR 失败"本身作为贡献 (SR 适用判据)。
**解决**: 推荐 **(C) 接受**——凸包能 SR 失败 + 我们的 SNR/线性度判据分析,
本身就是有价值的科学贡献。强行让 SR 在凸包能上"成功"反而不诚实。

## 二、整体严苛评判

### 2.1 是否构成完整研究?
**是, 但有缺口**。完整研究需要: 问题定义 → 数据 → 方法 → 评估 → 验证 → 讨论。
我们: 问题(钙钛矿稳定性预测)✓ 数据(4914)✓ 方法(PACT-SR)✓ 评估(CV+LOEO+UQ+AD)✓
验证(候选+matbench)✓ 讨论(SR边界+PCRL负面)✓。
**缺口**: (1) 无 DFT 验证候选; (2) 单数据源; (3) 候选无实验合成。
**评判**: 作为计算材料学方法学论文 (非实验发现论文), **完整度合格**。

### 2.2 现实应用价值?
**中等**。应用价值体现在:
- ✓ 可解释方程可直接用于材料设计规则 (电负性差大→形成能低)
- ✓ 候选 LaAlO3 等被 matbench 独立收录
- ⚠️ 但无 DFT/实验验证, 候选可信度未独立确认
- ⚠️ 凸包能预测 (材料发现核心) R²=0.79, 距实用 (R²>0.9) 有差距
**评判**: 作为"方法学贡献+初步筛选工具"有价值; 作为"可直接指导实验"尚早。

### 2.3 创新性本质 (最严苛的问题)
**诚实评判**: 我们的"创新"本质是**组合现有技术**——conformal (Vovk 2005) +
SR (gplearn) + 残差框架 (PGML) + LOEO。**没有单一组件是原创的**。

**但组合方式有价值**:
- ✅ 把 SR 方程嵌入残差框架 + conformal UQ + LOEO 外推评估, **据调研无先例**
- ✅ SR 适用边界的定量判据 (SNR/线性度) 是**新的分析贡献**
- ✅ LOEO 实验揭示"物理层价值在外推"——这个发现**有方法学意义**
**评判**: 创新性属于"**有价值的工程组合 + 方法学分析**", 非理论突破。
对三区 (Computational Materials Science) **足够**; 对二区顶刊 (npj) **偏弱**。

### 2.4 算法层面构建质量
**良好, 有改进空间**:
- ✓ 单一管线 (PACT-SR), 概念自洽
- ✓ CV 内独立 fit, 无泄露 (debug 验证)
- ✓ 物理层可插拔 (PGMLModel 架构)
- ⚠️ phys_idx_global 全局变量 (代码债)
- ⚠️ SR stopping_criteria=0 + generations=40 是手动调的, 无系统超参搜索
- ⚠️ 多 seed 协议未对 SR 做 (只有 CV 单 seed)

### 2.5 统计指标专业且有效?
**专业, 中上水平**:
- ✓ R²/MAE/RMSE/MRE/MaxErr + bootstrap CI
- ✓ PICP/MPIW/ECE (不确定性校准)
- ✓ σ-误差 Pearson + p 值
- ✓ F1/AUC/AUC-PR/DAF/EF (分类)
- ⚠️ Wilcoxon 5-seed 无分辨率 (p≥0.0625)——**这是真实统计极限, 论文须诚实写**
- ⚠️ LOEO 外推 R² 绝对值偏低 (0.42), 应报"相对优势"非绝对值

## 三、明显提升点 + 可实施性评估

### 提升1 (高ROI, 已验证): 重定位物理层价值 = 外推
**做什么**: 论文主表同时报 CV R² 和 LOEO R², 强调"物理层提升外推稳健性"。
**效果**: 直接解决"物理层没提精度"的质疑, 且有 LOEO 实验支撑。
**成本**: 零 (实验已做)。
**ROI**: ★★★★★

### 提升2 (高ROI, 可实施): 符号集成解决非唯一性
**做什么**: 跑 10 次 SR (不同 seed), 统计物理量在公式中的出现频率, 报"共识特征"。
**效果**: 解决问题③ (5 折不一致), 把"非唯一"从缺陷转为"集成共识"优势。
**成本**: ~30 min 计算。
**ROI**: ★★★★★

### 提升3 (中ROI): 丰富 SR 算子 + 帕累托
**做什么**: function_set 加 sin/exp/tanh, 扫描, 看形成能 R² 是否 0.41→0.55+。
**效果**: 可能提升 SR 物理层 R² (问题②), 但可解释性下降。
**成本**: ~20 min, 但需重新评估可解释性。
**ROI**: ★★★

### 提升4 (中ROI): 凸包能用分类 SR
**做什么**: 把凸包能阈值化为稳定/不稳定, 用 SR 学分类边界。
**效果**: 可能部分解决问题④, 但凸包能噪声大, 收益不确定。
**成本**: ~1h 改写。
**ROI**: ★★

### 提升5 (低ROI/高成本): 候选 DFT 验证
**做什么**: 对 2-3 个候选做单点 DFT (需 VASP/Quantum ESPRESSO)。
**效果**: 大幅提升应用价值与可信度。
**成本**: 本科生单机不现实 (需 HPC + DFT 经验)。
**ROI**: ★ (受资源限制)

## 四、最终严苛结论

**问: SR 物理层是否真正构成创新?**
答: **是, 但属于"工程组合创新 + 方法学分析", 非理论突破。** 对三区足够。
关键支撑: (1) 组合方式据调研无先例; (2) SR 适用判据是新分析贡献;
(3) LOEO 实验揭示物理层外推价值 (本次审稿新发现)。

**问: 现在能投三区吗?**
答: **能, 且比 V2 更稳。** 提升1 (LOEO 重定位) + 提升2 (符号集成) 是必做的两项
(零/低成本, 直接解决核心质疑)。做完后, 论文叙事变为:
"SR 物理层在外推下优于纯 ML (LOEO 证据) + 提供可解释方程 (符号集成共识)
+ 适用边界判据 (SNR/线性度)"——这是一条**诚实、有证据、有深度**的创新主线。

**问: 最大风险?**
答: 审稿人可能质疑"创新只是组合"。缓解: 强调 SR 适用边界判据 + LOEO 外推发现
这两个"分析贡献", 它们不是简单的技术组合。
