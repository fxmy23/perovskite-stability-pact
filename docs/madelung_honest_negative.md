# 形成能物理分解 (Madelung) — 诚实负面结果

**日期**: 2026-06-16
**结论**: 简单Madelung近似(理想立方+固定电荷)无预测力 (R²=-0.0007)。
诚实记录, 转向其他方向。

## 实验
PACT-Madelung: E_f = α·E_Madelung(解析) + ML残差

## pre-debug发现
Madelung能量 vs 形成能 Pearson r=0.012 (几乎无关), 线性映射R²=0.0001。

## 根因 (诚实)
我的Madelung近似用了:
- 固定电荷 z_eff²=12 (对所有ABO3相同)
- Madelung常数 M=1.716 (理想立方固定)
- 唯一变化量: 晶格常数 a≈2(r_B+r_O)
→ 结果: U只随r_B变化, 但r_B与形成能相关性弱(r=0.012)

**真正的Madelung能量需要**:
1. 实际DFT结构的Madelung常数 (随畸变变)
2. 实际Bader电荷 (非固定+2/+4)
3. Ewald求和 (需结构信息)
这些我们都缺 (只有组成描述符, 无结构)。

## 为什么之前的SR/KRR物理层有R²=0.43/0.79而Madelung没有
- SR/KRR用了**14个物理特征的组合** (电负性+半径+容差因子+电子结构)
- Madelung只用**1个变量** (晶格常数, 因为电荷固定)
- 物理特征的多样性 (尤其电负性) 才是预测力的来源, 非单一Madelung项

## 教训
1. 物理分解需要**足够多的物理自由度**, 固定电荷的Madelung太退化
2. 无结构信息时, 描述符ML(用多物理特征)优于解析物理公式(单变量)
3. "理论创新"必须考虑数据约束 — 我们无结构, Madelung做不实

## 论文处理
诚实报告为"我们尝试了Madelung物理分解但简单近似无效,
因无DFT结构信息, Madelung常数/电荷无法准确计算。
退回到SR/KRR数据驱动物理层"。

## 替代方案 (若要做物理分解)
需要Bader电荷或DFT结构 → 超出本科生单机 (需DFT计算)。
放弃此方向, 转向GBDT Stacking (更可靠的R²提升)。
